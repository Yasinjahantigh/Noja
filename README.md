نوژا یک تجربه وب مینیمال و تراپی‌محور است. فضایی امن و بی‌سروصدا که به شما اجازه می‌دهد افکار پراکنده، خشم‌ها و دغدغه‌هایتان را بنویسید و تماشا کنید که چگونه هر جمله، در یک بوم تاریک و بی‌کران به یک ستاره‌ی درخشان تبدیل می‌شود و صورت فلکی اختصاصی شما را می‌سازد.

بدون دیتابیس، بدون سرور، کاملاً امن. فکرهای شما فقط در مرورگر خودتان پردازش می‌شوند و سپس در وسعت یک کهکشان دیجیتال آرام می‌گیرند.

## 🌌 ویژگی‌های اصلی

* **تولد ستاره‌ها از کلمات:** هر جمله‌ای که تایپ می‌کنید، با فشردن کلید اینتر به یک ستاره در بوم تعاملی تبدیل شده و با خطوطی ظریف به افکار قبلی متصل می‌شود.
* **هوش مصنوعی در کلاینت (100% Private):** بدون ارسال حتی یک بایت به سرور! نوژا با استفاده از یک سیستم تحلیل احساساتِ سبک در مرورگر، بار عاطفی کلمات (شادی، خشم، آرامش) را می‌سنجد و رنگ، درخشش و ابعاد ستاره‌ها را بر اساس آن شخصی‌سازی می‌کند.
* **صداهای زنده و واکنشی (Generative Audio):** برای خلق یک اتمسفر غوطه‌ورکننده، نیازی به دانلود فایل‌های صوتی سنگین نیست. صداهای پس‌زمینه کهکشانی و ملودیِ کلیدهای کیبورد شما، به صورت زنده توسط کدهای برنامه تولید می‌شوند.
* **خروجی هنری:** در پایان جلسه، می‌توانید از صورت فلکی و منظومه زیبایی که از افکارتان خلق کرده‌اید، یک تصویر باکیفیت خروجی بگیرید.

## 🛠️ زیرساخت و معماری فنی

این پروژه با تمرکز بر **حداکثر کارایی و صفر بودن هزینه‌های نگهداری (Serverless)** طراحی شده است. تمام بار پردازشی بر عهده دستگاه کاربر است تا بالاترین سطح حریم خصوصی حفظ شود.

* **هسته فرانت‌اند:** React + Vite (برای لود فوق‌سریع و مدیریت بهینه Stateها)
* **گرافیک و رندرینگ:** معماری مبتنی بر Canvas دو‌بعدی برای رندر روان لایه‌های کیهانی و محاسبه هندسیِ فواصل ستاره‌ها بدون افت فریم.
* **موتور صوتی:** بهره‌گیری از Web Audio API برای سنتز صدای زنده.
* **استقرار کاملاً خودکار:** میزبانی روی GitHub Pages که با هر بار پوش کردن، توسط GitHub Actions بیلد و منتشر می‌شود.

## 🧠 پشت پرده‌ی الگوریتم‌ها

* **جایگذاری ارگانیک:** الگوریتم ریاضیِ نوژا از روی‌هم‌افتادگی ستاره‌ها جلوگیری می‌کند. هر ستاره جدید، با محاسبه‌ی فاصله اقلیدسی، در زاویه‌ای تصادفی اما منطقی نسبت به ستاره‌های قبلی قرار می‌گیرد تا منظومه‌ای درختی و طبیعی شکل بگیرد.
* **سحابی‌های پارالاکس (Parallax):** با حرکت ماوس در صفحه، ذرات غبار کیهانی و لایه‌های پس‌زمینه با سرعت‌های متفاوتی جابه‌جا می‌شوند تا حس عمقِ سه‌بعدی به کاربر منتقل شود.

## 🚀 راه‌اندازی محلی (Local Development)

برای اجرای نوژا روی سیستم خودتان و مشارکت در توسعه، مراحل زیر را دنبال کنید:

```bash
# ۱. کلون کردن مخزن
git clone https://github.com/Yasinjahantigh/Noja.git

# ۲. ورود به دایرکتوری پروژه
cd noja

# ۳. نصب پکیج‌ها
npm install

# ۴. اجرای سرور توسعه (Vite)
npm run dev
````

> **نکته:** از آنجایی که پروژه وابستگی به هیچ بک‌اندی ندارد، بلافاصله پس از اجرا در مرورگر شما با بالاترین سرعت آماده به کار است.
